# Tanstack Start + Cloudflare + Shadcn + Bun

## Getting Started

This is a basic template for a Cloudflare Worker with Tanstack Start, Shadcn, and TailwindCSS.

## Stack

- Cloudflare Wrangler
- Tanstack Start
- Shadcn
- TailwindCSS
- Oxc (`oxlint` + `oxfmt`)

### Install the dependencies

```
bun install
```
